# 🌌 StoryForge AI — AI-Powered Storytelling Platform

StoryForge AI is a cinematic AI storytelling engine that transforms simple story ideas into rich, immersive narratives using multi-layer AI architecture.

Built for the **Agents League Hackathon**, it combines creative writing, AI direction, and cinematic storytelling into one seamless experience.

---

## 🚀 Live Demo
https://forge-words-unfold.lovable.app/

---

## ✨ Key Features

### 🎬 Forge Mode (Core Engine)
- AI-powered story generation from simple prompts
- Multiple genres: Adventure, Sci-Fi, Horror, Romance, Fantasy, Mystery
- Tone control (Dark, Happy, Mysterious, Inspirational)
- Story length customization
- Real-time typewriter storytelling effect

---

### 🧠 Director Agent (Advanced AI Pipeline)
A multi-stage AI system that simulates a real “movie director” workflow:

1. Creative Blueprint
   - Story analysis
   - Genre recommendation
   - Character design
   - Plot structure

2. Story Generation
   - Full narrative creation using AI

3. Critique & Improvements
   - AI reviews story quality
   - Suggests improvements

> Optimized from 7 AI calls → 3 AI calls for free-tier efficiency.

## ⚙️ Tech Stack

- Frontend: React / TypeScript
- Backend: Node.js / API Routes
- AI: Gemini via Lovable AI Gateway
- Storage: LocalStorage + cloud session handling
- Styling: Custom cinematic UI system

---

## 🧠 AI Architecture

StoryForge uses a hybrid AI pipeline:

- Forge Mode → Single-call story generation
- Director Mode → Multi-stage AI reasoning pipeline
- Transform Engine → Story rewriting (tone, ending, format)

---

## ⚠️ Known Limitation

Director Agent may occasionally hit **AI rate limits (429)** due to free-tier API constraints.

This has been mitigated by:
- Reducing AI calls from 7 → 3
- Sequential execution
- Retry/backoff strategy
