# ✦ StoryForge AI — Craft Worlds With Words

> Microsoft Agents League Hackathon 2026 · Creative Apps Track

## What It Does
StoryForge AI transforms a simple premise into a rich narrative instantly — no backend or API key required.

## Features
- 6 Story Genres — Adventure, Sci-Fi, Horror, Romance, Fantasy, Mystery
- 4 Narrative Tones — Cinematic, Lyrical, Gritty, Whimsical
- 3 Story Lengths — Flash, Short, Epic
- Typing Animation — stories appear word by word
- Export — Copy, Download, Share
- 100% Offline — runs from a single HTML file

## Built With
- HTML5 + CSS3 + Vanilla JavaScript
- GitHub Copilot (AI-assisted development)

## Quick Start
Open index.html in any browser — or run:
python -m http.server 8000
